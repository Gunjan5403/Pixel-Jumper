# 🎮 Pixel Jumper

Pixel Jumper is a 2D browser-based platformer game built using **Firebase Studio** with a vibe-coding workflow.  
The game is developed using **Next.js, TypeScript, and HTML5 Canvas**, and delivers a smooth and responsive arcade experience.

## 🎮 Gameplay

In Pixel Jumper, the player:

- Runs and jumps across platforms
- Collects coins to increase score
- Defeats enemies by jumping on them
- Avoids enemy collisions
- Reaches the flag to win the level
