# **App Name**: Pixel Jumper

## Core Features:

- Player Movement: Enables left, right, and jump movements with realistic physics and collision detection.
- Enemy AI: Implements simple walking enemies that can be defeated by jumping on them.
- Collectibles System: Adds coins with sound and animation effects when collected.
- Score and Health System: Tracks the player's score, health/lives, and displays it in the HUD.
- Win Condition: Allows the player to win by reaching a flag or end door at the end of the level.
- Game Over Screen: Presents a Game Over screen with a restart option when the player loses all lives.
- Parallax Background: Generates a subtle parallax effect for the background to enhance the visual experience.

## Style Guidelines:

- Primary color: Bright cyan (#42cbf5) to give the game a cheerful and energetic mood.
- Background color: Very light cyan (#eaf8fd) to not distract the user, and make the primary color stand out.
- Accent color: Complementary light yellow (#f5e342), for interactive elements, warnings and highlights.
- Body and headline font: 'Inter', sans-serif, to achieve a modern, machined, objective, neutral look.
- Use simple, rounded icons for HUD elements (score, lives, coins).
- Responsive design to adapt to different screen sizes (mobile-friendly).
- Smooth transitions and animations for UI elements and gameplay actions (e.g., collecting coins, defeating enemies).
